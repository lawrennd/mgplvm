<h1>Mixtures of GP-LVM Software</h1>

<p>This page describes examples of how to use the fast Gaussian process latent variable model Software (MGPLVM).

RELEASEINFORMATION

This is the first release associated with the 2007 NIPS Submission.

<h2>Examples</h2>

<h3>Oil Data</h3>

<p>The `oil data' is commonly used as a bench mark for visualisation algorithms. For more details on the data see <a href="http://www.ncrg.aston.ac.uk/GTM/3PhaseData.html">this page</a>.

<p>We first consider the performance of the algorithm at relatively low numbers of data points, taking 100 points from the full data set.

<h4>Mixtures of Principal Coordinate Analysers</h4>

<p>The first example is run using <code>demOil1001.m</code>. It is a mixture of principal coordinate analysers (i.e. a mixture of GP-LVMs with linear kernels) using a gating network in the latent space. The model is initialised with 20 components. The results are shown in the figure below.

<p><center><img src="demOil1001.png" width="49%"><img src="demOil1001Cluster.png" width="49%"><br>
Mixtures of GP-LVM on the reduced oil data using <i>Left</i>: linear kernels (Principal Coordinate Analysis, <code>demOil1001</code>) and <i>Right</i>: the assignments of the points to clusters.</center>
<p><center><img src="demOil1002.png" width="49%"><img src="demOil1002Cluster.png" width="49%"><br>
Mixtures of GP-LVM on the reduced oil data using <i>Left</i>: linear kernels (Principal Coordinate Analysis, <code>demOil1001</code>) and <i>Right</i>: the RBF kernel (<code>demOil1002</code>). </center>

<p>The second example replaces the linear kernels with radial basis function kernels. This can be run using <code>demOil1002.m</code>.
